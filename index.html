My Skills

Responsive Design

Because today's web is “mobile-friendly”, I am able to adapt a website to all resolutions, from the smartphone to the computer screen, via the tablet. Responsive Design allows the site to be viewed in a suitable format from any device.

Web Design

From a creative brief or a specification, I can meet the needs of users by creating attractive and intuitive interfaces. I like to organize my work by respecting the logical stages of web design, from the project tree to graphic creation, until its development.

Web development

I use HTML5, CSS3, React and JavaScript. I carry out a technological watch daily in order to keep myself informed of the latest developments in front-end development: JavaScript, Angular, React, View ... So I make sure to learn new techniques to optimize my work.
